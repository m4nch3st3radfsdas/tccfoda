Nível Básico

Módulo I - Introdução ao Word

Curso ideal para iniciantes sem experiência prévia em Microsoft Word. Aprenda os conceitos fundamentais que servirão como base os demais módulos.

Módulos:

  • O que é o Microsoft Word e para que serve
  • Conhecendo a interface: faixa de opções, barra de ferramentas, área de trabalho
  • Criar, salvar e abrir documentos
  • Diferentes formatos de arquivos (.docx, .pdf, .rtf)

Duração: 4 horas


Módulo II - Edição de texto

Aprenda a criar suas primeiras páginas web com HTML e CSS. Curso prático com diversos exercícios e projetos reais.

Módulos:

  • Inserir e apagar texto
  • Selecionar, copiar, recortar e colar
  • Usar "Desfazer" e "Refazer"
  • Localizar e substituir palavras

Duração:2 Horas

Módulo III - Formatação de Texto

Aprenda a criar suas primeiras páginas web com HTML e CSS. Curso prático com diversos exercícios e projetos reais.

Módulos:

  • Alterar fonte, tamanho e cor
  • Negrito, itálico, sublinhado
  • Alinhamento (esquerda, centralizado, direita, justificado)
  • Espaçamento entre linhas e entre parágrafos
  • Uso correto de maiúsculas/minúsculas
  • Formatação de parágrafos com recuo

Duração: 1 Hora

Nível Intermediário

JavaScript Completo

Domine a linguagem de programação mais utilizada na web. Aprenda a criar sites interativos e dinâmicos com JavaScript.

Módulos:

  • Sintaxe e estruturas de controle
  • Funções e escopo
  • Manipulação do DOM
  • Eventos e listeners
  • Requisições assíncronas
  • Projeto: aplicativo de tarefas

Duração: 8 semanas

Professor: Pedro Santos

Banco de Dados SQL

Aprenda a modelar, criar e gerenciar bancos de dados relacionais utilizando SQL. Curso com muita prática e exercícios.

Módulos:

  • Modelagem de dados
  • Criação de tabelas e relacionamentos
  • Consultas SELECT avançadas
  • Inserção, atualização e exclusão de dados
  • Índices e otimização

Duração: 7 semanas

Professor: Ana Costa

Nível Avançado

Desenvolvimento Full Stack

Torne-se um desenvolvedor completo, dominando tanto o front-end quanto o back-end. Curso intensivo para quem já tem conhecimentos em programação.

Módulos:

  • React.js para front-end
  • Node.js e Express para back-end
  • MongoDB e bancos NoSQL
  • Autenticação e segurança
  • APIs RESTful
  • Projeto: aplicação web completa

Duração: 12 semanas

Professor: Carlos Mendes

DevOps e Cloud Computing

Aprenda as melhores práticas de DevOps e implantação em nuvem. Domine ferramentas como Docker, Kubernetes e AWS.

Módulos:

  • Integração e entrega contínua (CI/CD)
  • Conteinerização com Docker
  • Orquestração com Kubernetes
  • AWS e serviços em nuvem
  • Monitoramento e observabilidade
  • Projeto: pipeline de implantação

Duração: 10 semanas

Professor: Luciana Alves
